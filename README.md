# Primer Avance - Gestión de Inventarios

Proyecto académico para la materia **Estructuras de Datos (SOFT-10)**.

Implementa una **lista enlazada simple** en Java para la gestión de productos con operaciones CRUD, búsqueda, y generación de reportes. Interfaz interactiva por consola.

## Estructura

- `src/` — Código fuente Java
- `images/` — Imágenes de productos

## Ejecución

Compilar y ejecutar desde `src/`:

```bash
javac src/*.java -d out && java -cp out Main
```
